# 🤖 AI Resume Analyzer

An AI-powered Resume Analysis and Recruitment Platform built using **Streamlit**, **Google Gemini AI**, and **SQLite**.

The application serves two types of users:

- 👨‍🎓 Students
- 👨‍💼 Recruiters

Students can analyze their resumes, receive ATS feedback, identify missing skills, compare against job descriptions, and generate personalized improvement plans.

Recruiters can upload multiple resumes, automatically evaluate candidates using AI, rank applicants based on customizable criteria, shortlist candidates, and export shortlisted candidates for further hiring.

---

## 🚀 Features

### 👨‍🎓 Student Portal

- ATS Resume Analysis
- Resume Completeness Check
- Skill Detection
- Missing Skill Identification
- Job Description Matching
- Keyword Matching Analysis
- Resume Section Scoring
- Resume Strength Evaluation
- Personalized 30-Day Improvement Plan
- PDF Report Generation

### 👨‍💼 Recruiter Portal

- Upload Multiple Resumes
- Automatic Resume Validation
- AI Candidate Evaluation
- Candidate Ranking
- Dynamic "What-if" Ranking Analysis
- AI Candidate Summaries
- Shortlist / Remove Shortlist
- Export Shortlisted Candidates (CSV)
